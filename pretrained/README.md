# Pretrained Models

This directory should contain the pretrained model weights.

## Required Files

### TPGD Model
- `tpgd_ViT-B-32.pt` - TPGD ViT-Base model (download from the original repository or train your own)

### Universal Image Restoration Model
- `universal-ir.pth` - Universal image restoration model (download from the original repository or train your own)

## Download Links

Please download the pretrained models from the original paper's repository and place them in this directory with the exact filenames above.

## Note

All configuration files have been updated to use relative paths pointing to this directory.
